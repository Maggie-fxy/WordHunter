二、游戏首页（进入页）视觉元素设计
首页定位为 “狩猎模式入口 + 核心玩法展示 + 视觉吸引力” 三合一，遵循Playful Soft Neubrutalism风格，结合产品功能逻辑，拆解为以下视觉模块：
1. 整体风格基调
背景：全屏填充#FFFDF5（米白色），无多余纹理，保持简洁干净，突出核心元素。
边框与线条：所有模块使用 粗黑实线（2-3px） 勾勒边框，边角做8px圆角处理，符合新粗野主义 “硬朗又可爱” 的特质。
色彩配比：主色#E57373（三文鱼粉）用于核心按钮 / 标题；辅助色#FFD54F（向日葵黄）用于高亮提示 / 勋章点缀；文字色#2D2D2D（深灰），避免纯黑刺眼。
2. 顶部区域（Header）
元素	设计细节	交互逻辑
Logo 展示区	左侧放置游戏 Logo（建议用提示词 1 的精灵款），尺寸60x60px；右侧并列显示 游戏全称：上方英文Word Hunter（粗体圆角字体，字号 20px），下方中文单词宝可梦（圆润卡通字体，字号 16px）	点击 Logo 无交互（MVP 阶段），仅作品牌展示
个人入口	右上角放置 卡通头像框（尺寸50x50px），默认头像为 “戴小帽子的小猎人”，头像右下角叠加#FFD54F圆形小标（可显示未读复习提醒，MVP 阶段简化为静态）	点击头像直接跳转 Profile Mode（个人成就页）
3. 中部核心区域（Main Content）
这是首页的视觉焦点，直接传递核心玩法，分为上下两部分：
（1）单词任务卡片
容器样式：300x120px矩形卡片，#FFFDF5底色 + 粗黑边框，顶部加#E57373色条（高度 8px）。
内部元素：
顶部居中：当前目标单词（如Apple），加粗字号 24px，带 发音按钮（喇叭图标，#E57373色，点击播放单词读音）。
底部居中：单词配图（MVP 阶段用卡通简笔画，后续可替换为用户历史捕获图），尺寸60x60px。
交互：点击发音按钮播放音效，无其他交互。
（2）卡槽展示区
布局：2×3网格布局（适配 6 个待收集单词位），每个卡槽尺寸80x80px。
样式：未收集时为 灰色占位框（带虚线边框 + 问号图标）；已收集时显示用户拍摄的抠图战利品（如苹果照片），卡槽边框变为#E57373色。
视觉提示：卡槽下方显示 收集进度文字（如已收集2/6），字号 14px，#2D2D2D色。
4. 底部区域（Footer）
核心按钮：“START HUNTING 去捕获” 悬浮式大按钮，尺寸280x70px，#E57373纯色填充，白色粗体字（字号 22px），按钮加轻微阴影（drop-shadow(3px 3px 0 #2D2D2D)），强化点击欲望。
交互：点击后触发相机模块，滑入相机取景框，进入 Hunter Mode。
模式切换入口：按钮下方放置两个小按钮（120x40px），左右排列：
左侧：“复习营地”（#FFD54F底色），点击跳转 Review Mode，按钮上显示待复习单词数（如3）。
右侧：“成就勋章”（#FFFDF5底色 + 边框），点击跳转 Profile Mode 的勋章墙。
5. 点缀元素（增强趣味性）
钻石图标：在卡槽区右上角，放置一个小钻石图标（#FFD54F色），旁边显示当前钻石数（如💎 12），强化奖励反馈。
动态小动画：使用 Framer Motion 添加简单动效 ——Logo 精灵轻微上下跳动、核心按钮呼吸式缩放、已收集卡槽的图片轻微旋转，提升页面活力（MVP 阶段简化为 CSS 动画）。
